📋 Tema 04: Menus e Validações de Dados

Aprenda a criar menus interativos e validar entradas do usuário

🎯 O que vamos aprender?

Menus são a porta de entrada para qualquer sistema. Eles permitem que o usuário escolha entre várias opções de forma organizada.

Validações garantem que o programa não quebre quando o usuário digitar algo errado.

O comando ESCOLHA-CASO é a melhor forma de criar menus no Portugol!

🧮 Calculadora Simples
✅ Menu com ESCOLHA-CASO e pausa com ENTER
programa
{
    funcao inicio()
    {
        inteiro opcao
        real num1, num2, res
        cadeia pausa
        
        faca
        {
            escreva("\n===== CALCULADORA =====\n")
            escreva("1 - SOMA\n")
            escreva("2 - SUBTRAÇÃO\n")
            escreva("3 - SAIR\n")
            escreva("=======================\n")
            escreva("Opção: ")
            leia(opcao)
            
            escolha(opcao)
            {
                caso 1:
                    escreva("Digite o 1º número: ")
                    leia(num1)
                    escreva("Digite o 2º número: ")
                    leia(num2)
                    res = num1 + num2
                    escreva("\nResultado: ", num1, " + ", num2, " = ", res, "\n")
                pare
                
                caso 2:
                    escreva("Digite o 1º número: ")
                    leia(num1)
                    escreva("Digite o 2º número: ")
                    leia(num2)
                    res = num1 - num2
                    escreva("\nResultado: ", num1, " - ", num2, " = ", res, "\n")
                pare
                
                caso 3:
                    escreva("\nPrograma encerrado!\n")
                pare
                
                caso contrario:
                    escreva("\n❌ OPÇÃO INVÁLIDA! Digite 1, 2 ou 3.\n")
            }
            
            se (opcao != 3)
            {
                escreva("\nPressione ENTER para continuar...")
                leia(pausa)
            }
            
        } enquanto (opcao != 3)
    }
}

🏦 Caixa Eletrônico
✅ Menu com validação de saldo
programa
{
    funcao inicio()
    {
        inteiro opcao
        real saldo = 1000.00
        real valor
        cadeia pausa
        
        faca
        {
            escreva("\n===== CAIXA ELETRÔNICO =====\n")
            escreva("1 - CONSULTAR SALDO\n")
            escreva("2 - SACAR\n")
            escreva("3 - DEPOSITAR\n")
            escreva("4 - SAIR\n")
            escreva("===========================\n")
            escreva("Opção: ")
            leia(opcao)
            
            escolha(opcao)
            {
                caso 1:
                    escreva("\n💰 Saldo atual: R$ ", saldo, "\n")
                pare
                
                caso 2:
                    escreva("Valor para saque: R$ ")
                    leia(valor)
                    se (valor > saldo)
                    {
                        escreva("\n❌ Saldo insuficiente!\n")
                    }
                    senao
                    {
                        saldo = saldo - valor
                        escreva("\n✅ Saque realizado! Novo saldo: R$ ", saldo, "\n")
                    }
                pare
                
                caso 3:
                    escreva("Valor para depósito: R$ ")
                    leia(valor)
                    saldo = saldo + valor
                    escreva("\n✅ Depósito realizado! Novo saldo: R$ ", saldo, "\n")
                pare
                
                caso 4:
                    escreva("\n👋 Obrigado por usar nosso caixa!\n")
                pare
                
                caso contrario:
                    escreva("\n❌ Opção inválida! Digite 1, 2, 3 ou 4.\n")
            }
            
            se (opcao != 4)
            {
                escreva("\nPressione ENTER para continuar...")
                leia(pausa)
            }
            
        } enquanto (opcao != 4)
    }
}

📌 Resumo: Menus e Validações

  • ESCOLHA-CASO é a estrutura ideal para criar menus com opções fixas
  • Sempre use "pare" após cada caso para evitar execução em cascata
  • "CASO CONTRARIO" é o "else" do ESCOLHA-CASO - use para tratar opções inválidas
  • faca...enquanto mantém o menu em loop até o usuário escolher sair
  • NÃO use "limpa()" para que o resultado fique visível até o ENTER
  • Use uma variável de pausa (ex: cadeia pausa) com leia(pausa) para aguardar o ENTER
💡 Dica Importante:
O comando ESCOLHA-CASO não aceita operadores lógicos como ">" ou "<".
Use SE/SENÃO para validações de intervalo e ESCOLHA-CASO para opções exatas.
