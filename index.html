📌 Preparação para o Envio:
Crie um repositório chamado tema4 e salve dentro dele os arquivos:
  • arquivo.por (Seu código do Desafio Portugol)
  • index.html (Seu código do Desafio WebApp)
🚀 Passo a passo para ativar o link:
  1. No repositório tema4, clique em Settings (Configurações).
  2. No menu lateral, clique em Pages.
  3. Em Branch, selecione main e a pasta / (root).
  4. Clique em Save e aguarde o deploy.
  5. Acesse seu projeto no link em “Your site is live at...”.
